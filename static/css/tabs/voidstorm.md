<h1>Активные бури бездны</h1>

Для просмотра информации об активных бурях бездны на сайте есть специальная команда.

<br></br>

Синтаксис выглядит следующим образом:

_<h4>/worldstate VoidStorms</h4>_

<br></br>

Первая часть команды "_/worldstate_" обращается к соответствующей базе данных, а через пробел далее прописывается _VoidStorms_.

Данные со временем могут меняться, бури бездны не постоянны.

<br></br>

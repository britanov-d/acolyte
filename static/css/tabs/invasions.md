<h1>Активные вторжения</h1>

Для просмотра информации об активных вторжениях на сайте есть специальная команда.

<br></br>

Синтаксис выглядит следующим образом:

_<h4>/worldstate Invasions</h4>_

<br></br>

Первая часть команды "_/worldstate_" обращается к соответствующей базе данных, а через пробел далее прописывается _Invasions_.

Данные со временем могут меняться, вторжения имеют лимит времени.

<h1>Просмотр подробной статистики по аккаунту</h1>

Статистика собирает довольно много скрытой информации об аккаунте, для вывода статистика на сайте есть отдельная команда.

<br></br>

Синтаксис выглядит следующим образом:

_<h4>/profile id</h4>_

<br></br>

или

<br></br>

_<h4>/stats id</h4>_

<br></br>

Первая часть команды "_/stats_" или "_/profile_" обращается к соответствующей базе данных, а через пробел далее указываются данные профиля.

К сожалению, возможность обращаться к статистике напрямую через никнейм была ограничена в 2025 году, так что теперь для этого стоит использовать уникальный id.

<br></br>

Id можно найти здесь /Users/Username/AppData/Local/Warframe/EE.log. В этом файле ищите следующее поле:

![руководство по поиску айди профиля](/static/images/id-profile.png)

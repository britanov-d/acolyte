<h1>Активные разрывы бездны</h1>

Для просмотра информации об активных разрывах бездны на сайте есть специальная команда.

<br></br>

Синтаксис выглядит следующим образом:

_<h4>/worldstate ActiveMissions</h4>_

<br></br>

Первая часть команды "_/worldstate_" обращается к соответствующей базе данных, а через пробел далее прописывается _ActiveMissions_.

Данные со временем могут меняться, разрывы бездны не постоянны, каждый разрыв активен около полутора часов.
